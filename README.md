# VidTube Frontend

A stunning YouTube-inspired React frontend for the VidTube backend API.

## Features
- 🎥 Browse & watch videos
- 🔐 Register / Login
- ⬆️ Upload videos with thumbnail
- 👍 Like videos & comments
- 🔔 Subscribe to channels
- 📋 Playlists & Watch History
- 🔍 Search videos
- 📊 Channel profiles

## Setup & Run

```bash
npm install
npm run dev
```

Opens at: http://localhost:3000

> Make sure backend is running at http://localhost:7000
